Connected Component
  • Admin
  • Mainly responsible: Shuang Wu

When trying to realize the connected component algorithm, we first initialize the status of each vertex to be the identification number(To ensure there will not be the situation where 2 different components share the same component name). Then we go through lots of steps. In each step, every vertex update their status by comparing their own status with the specific number sent from one of their parent vertex, which is the parent's status. We choose the smaller one to replace the status of current vertex. Repeat the above steps until there is no vertex's status need to be changed.
